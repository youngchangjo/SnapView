SnapView 사용자 매뉴얼

이 문서는 SnapView를 처음 사용하는 사용자가 설치부터 선별 작업까지 혼자 진행할 수 있도록 만든 실사용 안내서입니다.

1. SnapView가 무엇인가요?

  • SnapView는 macOS용 초고속 이미지 뷰어입니다.
  • 무거운 카탈로그 생성 없이 폴더/ZIP을 바로 열어 빠르게 감상하고 선별할 수 있습니다.
  • Lightroom 같은 편집 도구를 대체하기보다, 그 전에 빠르게 보고 고르는 단계에 집중합니다.

2. 이런 분께 추천합니다

  • 여행/일상/가족 사진을 많이 찍는 분
  • RAW 파일도 보지만 무거운 워크플로우가 부담스러운 분
  • 빠르게 훑고 좋은 컷만 남기고 싶은 분
  • 폴더 기반으로 간단히 관리하고 싶은 분

3. 설치와 첫 실행

3.1 설치

  1. 릴리즈에서 SnapView.dmg를 다운로드합니다.
  2. DMG를 열고 SnapView.app을 Applications로 드래그합니다.
  3. Applications에서 SnapView를 실행합니다.

3.2 처음 실행 시

  • macOS 보안 경고가 뜨면:
    1. 시스템 설정 -> 개인정보 보호 및 보안
    2. 실행 허용
    3. 앱 재실행
  • 첫 실행 화면에서 기본 이미지 앱 등록 제안을 볼 수 있습니다.
  • 원치 않으면 건너뛰어도 사용에 문제 없습니다.

4. 5분 빠른 시작

  1. 열기(Open)로 폴더 또는 ZIP 파일을 엽니다.
  2. Q/W/E/R로 뷰 모드를 바꿔 봅니다.
  3. 좌/우 화살표로 넘기며 확인합니다.
  4. 좋은 컷은 1~5로 별점, 불필요한 컷은 X로 Reject 표시합니다.
  5. 비교가 필요하면 C로 Compare 모드에서 1~4장 비교합니다.
  6. Delete로 삭제하거나 Cmd+C로 복사해 정리합니다.

5. 화면 구성 이해하기

  • 상단 툴바: 열기, 이전/다음, 뷰 모드 전환, 줌, 정보/퀵체크/히스토그램 토글, 선별 액션
  • 중앙 캔버스: 현재 이미지 표시
  • 모드별 보조 영역:
    • Grid: 썸네일 격자
    • Filmstrip/Manage: 하단 또는 사이드 영역에서 연속 탐색
    • 상세 EXIF/Quick Check 패널: 정보 분석용

6. 뷰 모드별 사용법

6.1 Main View ( W)

  • 한 장에 집중해서 확인할 때 사용합니다.
  • 0으로 Fit, 9로 100%를 빠르게 전환합니다.
  • I로 간단 정보 오버레이를 켜고 끌 수 있습니다.

6.2 Grid View ( Q)

  • 많은 사진을 한 번에 훑을 때 사용합니다.
  • 화살표/마우스로 빠르게 이동하고 여러 장을 선택할 수 있습니다.
  • =/-로 썸네일 크기를 조절해 밀도를 맞춥니다.

6.3 Filmstrip View ( E)

  • 메인 이미지 + 연속 스트립으로 흐름을 유지하며 볼 때 좋습니다.
  • 연사/유사 컷을 빠르게 넘기며 감으로 선별하기 좋습니다.

6.4 사진관리모드 Manage ( R)

  • 폴더/필터/선택 상태를 보며 정리할 때 사용합니다.
  • 별점/플래그/컬러 라벨을 적용하고 필터링해 결과를 정돈합니다.

7. 선별 도구 (가장 많이 쓰는 기능)

7.1 별점

  • 1~5: 별점 부여
  • 0 또는 \(원화키 포함): 별점 해제

7.2 Pick / Reject / Unflag

  • P: Pick
  • X: Reject
  • U: Unflag

7.3 컬러 라벨

  • Cmd+1~5: 빨강/노랑/초록/파랑/보라
  • Cmd+6: 해제

7.4 복사/삭제

  • Cmd+C: 선택 항목 복사
  • Cmd+V: 붙여넣기
  • Delete 또는 D: 삭제
  • Cmd+Delete: 즉시 삭제(주의)

8. Quick Check와 Compare 사용법

8.1 Quick Check

  • K로 Quick Check 세트를 켭니다.
  • 점수와 경고 원인(초점/노출/눈감음/구도 등)을 확인합니다.
  • Quick Check는 보조 도구이며 최종 선택은 사용자가 합니다.

8.2 Compare 모드

  • C로 Compare 모드 진입/종료
  • 1~4장을 나란히 비교해 한 장을 고릅니다.
  • 확대해서 디테일까지 확인할 수 있습니다.

9. ZIP 파일 사용 시 주의점

  • ZIP 내부 이미지도 일반 폴더처럼 탐색할 수 있습니다.
  • ZIP은 구조 특성상 폴더보다 느릴 수 있습니다.
  • ZIP 환경에서는 일부 정리 기능 제약이 있을 수 있습니다.
  • 문서 기준으로 별점 저장은 ZIP에서 제한됩니다.

10. 성능을 높이는 방법

  1. 대용량/ZIP 탐색이 느리면 디코딩 모드를 속도 우선으로 변경합니다.
  2. 그리드에서는 썸네일 크기를 적당히 줄여 한 화면 밀도를 높입니다.
  3. 필요한 오버레이만 켜고( I, K, H) 불필요한 패널은 끕니다.
  4. 느린 외장 저장장치에서는 먼저 작은 폴더 단위로 정리합니다.

11. 자주 쓰는 단축키 요약

  • 뷰 전환: Q / W / E / R
  • 비교 모드: C
  • 이동: 좌/우, 처음/끝 상/하
  • 점프 이동: Cmd+좌/우(10장), Cmd+Shift+좌/우(50장)
  • 줌: = / -, Fit 0, 100% 9
  • 정보 토글: I(정보), K(퀵체크 세트), H(히스토그램)
  • 선별: 1~5, 0, P, X, U, Cmd+1~6
  • 작업: Cmd+C, Cmd+V, Delete, Cmd+O

12. 추천 워크플로우 (실전)

12.1 여행/일상 사진 빠른 정리

  1. Grid( Q)에서 전체를 훑습니다.
  2. Reject( X)를 먼저 표시해 불필요 컷을 줄입니다.
  3. Main( W) 또는 Filmstrip( E)으로 좋은 컷에 별점 3~5를 줍니다.
  4. 애매한 컷은 Compare( C)로 2~4장만 비교합니다.
  5. 최종 선택본을 복사/이동해 보관 폴더로 정리합니다.

12.2 RAW+JPG 혼합 촬영 정리

  1. 폴더를 열고 빠르게 탐색합니다.
  2. Quick Check( K)로 문제 컷을 빠르게 찾습니다.
  3. 별점/플래그로 선택 결과를 만든 뒤 Manage( R)에서 필터로 점검합니다.

13. 문제 해결 (FAQ)

Q1. 앱이 열리지만 파일이 안 보입니다

  • 지원 포맷인지 확인합니다.
  • 파일 권한 문제면 폴더를 다시 선택해 권한을 허용합니다.

Q2. 키보드 단축키가 안 먹습니다

  • 앱 창이 포커스를 가지고 있는지 확인합니다.
  • 다른 입력창(텍스트 필드)에 커서가 있으면 먼저 해제합니다.

Q3. ZIP이 느리거나 실패 메시지가 나옵니다

  • 손상 ZIP인지 확인합니다.
  • 다시 압축하거나 파일을 다시 받아 테스트합니다.
  • 디코딩 모드를 속도 우선으로 낮춰봅니다.

Q4. 아이콘/버전 표기가 이상합니다

  • 앱 재실행 후 확인합니다.
  • 개발 빌드 사용 시 클린 빌드 후 다시 실행합니다.

14. 업데이트

  • 앱 메뉴에서 업데이트 확인(Check for Updates)를 실행할 수 있습니다.
  • 최신 릴리즈는 SnapView.dmg 기준으로 배포됩니다.

SnapView User Manual

This is a practical guide designed to help new users navigate SnapView from installation to selection independently.

1. What is SnapView?

  • SnapView is a blazing-fast image viewer for macOS.
  • Open folders or ZIP files instantly to view and select photos without creating heavy catalogs.
  • Rather than replacing editing tools like Lightroom, it focuses on the quick viewing and culling stage before editing.

2. Recommended For

  • People who take many travel, daily life, or family photos.
  • Users who shoot RAW but find heavy workflows burdensome.
  • Those who want to scan quickly and keep only the best shots.
  • Users who prefer simple folder-based management.

3. Installation and First Run

3.1 Installation

  1. Download SnapView.dmg from the releases page.
  2. Open the DMG and drag SnapView.app to Applications.
  3. Launch SnapView from Applications.

3.2 First Run

  • If a macOS security warning appears:
    1. Go to System Settings -> Privacy & Security.
    2. Allow execution.
    3. Relaunch the app.
  • On the first run, you may see a suggestion to set it as the default image app.
  • You can skip this if you prefer; it won't affect usage.

4. 5-Minute Quick Start

  1. Use Open to open a folder or ZIP file.
  2. Switch view modes with Q/W/E/R.
  3. Navigate using Left/Right Arrow keys.
  4. Rate good shots with 1~5, or mark unwanted ones with X (Reject).
  5. If you need to compare, press C for Compare Mode (1-4 images).
  6. Use Delete to remove or Cmd+C to copy and organize.

5. Understanding the Interface

  • Top Toolbar: Open, Prev/Next, View Mode Switch, Zoom, Info/Quick Check/Histogram Toggles, Selection Actions.
  • center Canvas: Displays the current image.
  • Mode-Specific Areas:
    • Grid: Thumbnail grid.
    • Filmstrip/Manage: Continuous navigation bar at bottom or side.
    • Detail EXIF/Quick Check Panel: For information analysis.

6. How to Use View Modes

6.1 Main View ( W)

  • Use when focusing on a single image.
  • Quickly switch between Fit (0) and 100% (9).
  • Toggle simple info overlay with I.

6.2 Grid View ( Q)

  • Use to scan many photos at once.
  • Navigates quickly with arrows/mouse and supports multi-selection.
  • Adjust thumbnail size with =/- to fit your screen.

6.3 Filmstrip View ( E)

  • Good for viewing flow with Main Image + Continuous Strip.
  • Ideal for quickly skimming through burst/similar shots by feel.

6.4 Manage Mode ( R)

  • Use when organizing with Folder/Filter/Selection views.
  • Apply Ratings/Flags/Color Labels and filter to tidy up results.

7. Culling Tools (Most Used Features)

7.1 Ratings

  • 1~5: Apply Star Rating.
  • 0 or \: Clear Rating.

7.2 Pick / Reject / Unflag

  • P: Pick.
  • X: Reject.
  • U: Unflag.

7.3 Color Labels

  • Cmd+1~5: Red/Yellow/Green/Blue/Purple.
  • Cmd+6: Clear.

7.4 Copy/Delete

  • Cmd+C: Copy selected items.
  • Cmd+V: Paste.
  • Delete or D: Delete.
  • Cmd+Delete: Delete Immediately (Caution).

8. Quick Check & Compare

8.1 Quick Check

  • Press K to toggle the Quick Check set.
  • Check scores and warning causes (Focus/Exposure/Eyes Closed/Composition).
  • Quick Check is an assist tool; the final choice is yours.

8.2 Compare Mode

  • Press C to enter/exit Compare Mode.
  • Compare 1-4 shots side-by-side to pick the best one.
  • Zoom in to check details.

9. Notes on ZIP Files

  • You can browse images inside ZIPs just like folders.
  • ZIPs may be slower than folders due to their structure.
  • Some organization features may be limited in ZIP environments.
  • Saving ratings to files is restricted in ZIPs.

10. Performance Tips

  1. If large files/ZIPs are slow, change Decoding Mode to Speed Priority.
  2. In Grid, reduce thumbnail size to increase density on one screen.
  3. Turn on only necessary overlays ( I, K, H) and close unused panels.
  4. On slow external drives, organize in smaller folders first.

11. Shortcuts Summary

  • View Switch: Q / W / E / R
  • Compare Mode: C
  • Move: Arrows, First/Last Up/Down
  • Jump: Cmd+Arrow (10), Cmd+Shift+Arrow (50)
  • Zoom: = / -, Fit 0, 100% 9
  • Toggle Info: I (Info), K (Quick Check), H (Histogram)
  • Cull: 1~5, 0, P, X, U, Cmd+1~6
  • Actions: Cmd+C, Cmd+V, Delete, Cmd+O

12. Recommended Workflow

12.1 Quick Organization for Travel/Daily

  1. Scan everything in Grid ( Q).
  2. Mark Rejects ( X) first to reduce clutter.
  3. Rate good shots 3~5 in Main ( W) or Filmstrip ( E).
  4. Compare ambiguous shots ( C) to pick the winner.
  5. Copy/Move final selections to an archival folder.

12.2 RAW+JPG Hybrid Workflow

  1. Open folder for quick browsing.
  2. Use Quick Check ( K) to spot technical issues fast.
  3. After Rating/Flagging, check results with filters in Manage ( R).

13. Troubleshooting (FAQ)

Q1. App opens but no files are visible

  • Check if the format is supported.
  • If it's a file permission issue, re-select the folder to grant permission.

Q2. Keyboard shortcuts aren't working

  • Check if the app window has focus.
  • If text cursor is in a search field, click away to focus the main window.

Q3. ZIP is slow or shows failure messages

  • Check if the ZIP is corrupted.
  • Re-compress or re-download the file.
  • Try lowering Decoding Mode to Speed Priority.

Q4. Icon/Version display is incorrect

  • Restart the app.
  • If using a dev build, clean build and relaunch.

14. Updates

  • Use Check for Updates in the app menu.
  • Latest releases are distributed via SnapView.dmg.
